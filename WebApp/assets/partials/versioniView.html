Versioni disponibili

Attenzione, l’App Fare Parole è ottimizzata per alcuni browser web e i dispositivi mobile più recenti. Nello specifico, per consultare l’App da PC si consiglia l’utilizzo del browser Chrome. Per consultare l’App da Tablet, si consigliano le versioni iOS 9.0 o superiori e Android 5.0 o superiori. Requisiti differenti possono causare malfunzionamenti nella consultazione.

Fare Parole 2 è disponibile in cinque versioni:

1 - Web App

La versione per web di Fare Parole è accessibile via internet all'indirizzo http://www.fareparole.it/fareparole2. Funziona su PC, Tablet e Smartphone.

2 - Chrome App

Fare Parole può essere installata come Chrome App. Il vantaggio di questa soluzione è che una volta installata non sarà più necessario utilizzare Internet, la Chrome App sarà pertanto disponibile anche off-line. Per installare la Chrome App scarichiamo Fare Parole dal link: http://www.fareparole.it/fareparole2/download/fareparole2.crx. Una volta scaricato il file, andiamo in Chrome, menu Impostazioni > Altri strumenti > Estensioni e trasciniamo all'interno della pagina il file scaricato. Da questo momento è possibile utilizzare la Chrome App anche off-line.

3 - Chrome App Portable

Fare Parole può essere copiata in una semplice chiavetta usb, o direttamente sul PC! Può essere utilizzata off-line e senza la necessità di alcuna installazione. Il file zip è scaricabile all'indirizzo: http://www.fareparole.it/fareparole2/download/fareparole2.zip. Una volta scaricato, estraiamo lo zip e lanciamo GoogleChromePortable.exe. Troveremo Fare Parole disponibile tra le Chrome App.

4 - Tablet Android - ( versione Android 4.0 o superiore)

Fare Parole può essere installata come App sul proprio tablet Android! Il vantaggio di questa soluzione è che una volta installata non sarà più necessario utilizzare Internet, l'App sarà pertanto utilizzabile anche off-line.
Per installare l'App su tablet Android andare all'indirizzo: http://www.fareparole.it/fareparole2/download/fareparole2.apk.
Alla comparsa delle notifica: "Questo tipo di file può danneggiare il tuo dispositivo. Vuoi conservare comunque?" Premere OK.
Andare nella cartella Download del proprio tablet e cliccare sul file fareparole.apk.
Se richiesto, cliccare su Impostazioni > Sicurezza > Sorgenti sconosciute. Confermare e continuare.
Cliccare infine su installa.
Al termine dell'installazione Fare Parole 2 sarà disponibile tra le applicazioni.

5 - iPad - ( versione 2 o superiore) - DISPONIBILE PROSSIMAMENTE

Fare Parole 2 può essere installata come App sul proprio iPad. Il vantaggio di questa soluzione è che una volta installata non sarà più necessario utilizzare Internet, l'App sarà pertanto utilizzabile anche off-line. Questa versione sarà disponibile nei prossimi giorni.
